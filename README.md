# Particle Orbit Theory

The motion of charged particles in electromagnetic fields. BSc Honours project (MT4599), University of St Andrews.

The Lorentz force equation is integrated with `scipy.integrate.solve_ivp` (RK45) and the numerical output is compared against analytic predictions from particle orbit theory. The same solver is applied across a progression of field configurations, from uniform fields up to a rotating tilted dipole representing a planetary magnetosphere.

## Repository layout

```
Applications/   Python scripts and core modules (run from here)
  orbit_ivp_core.py   Full Lorentz solver and guiding-centre extraction
  fields.py           Field library (uniform, gradient, dipole, tilted, rotating)
  guiding_centre.py   Guiding-centre equations of motion
  test01-test16.py    Test and simulation scripts — see Applications/README.md
  animate*.py         Animation scripts that produce GIFs
Figures/         Figures used in the report and animations (PNGs regenerated by scripts; GIFs tracked)
Presentation/    Project presentation (.pptx)
Results/         Validation data and numerical summary
```

## Setup

```bash
pip install -r requirements.txt
```

Python 3.10 or newer.

## Running

All scripts are run from inside `Applications/`:

```bash
cd Applications
python test01_uniformB_xy_orbit.py
```

PNG figures are written to `../Figures/` (not tracked in git — regenerate by running the scripts). GIF animations are tracked.

## Validation

Headline numbers from the test suite (see `Results/validation_summary.md` for the full breakdown):

- E × B drift speed matches theory exactly
- Gradient-B drift within 3 % of the analytic first-order formula
- Bounce period within 1 % of the analytic integral
- SI 10 keV electron mirror latitude accurate to 0.02 %
- Corotation E × B contribution recovered to 96.6 % of input Ω
